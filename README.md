# Mi Proyecto Web

## Descripción
Proyecto web simple para practica de Git

## Tecnologías
- HTML
- CSS  
- JavaScript
- Git

## Instalación
1. Clona el repo: `git clone URL`
2. Abre index.html en tu navegador

## Despliegue
### Servidor local (XAMPP)
- Copiar carpeta en htdocs
- Acceder vía localhost/mi-proyecto-web

### Hosting
- Subir archivos por FTP
- Configurar dominio

## Estructura

```
mi-proyecto-web/
├── css/
│ └── styles.css
├── js/
│ └── script.js
├── index.html
└── README.md
```
